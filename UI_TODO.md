# UI TODO 리스트

## 구현 완료 ✅
- ✅ **캐릭터 시트 (Character Sheet)**: 'C' 키로 스탯 포인트 배분 가능
- ✅ **내구도 경고**: 50% 이하 반짝임, 10% 이하 빨간색 표시
- ✅ **몬스터 HP 바**: 전투 중 몬스터 위에 HP 게이지 표시 (3초간)
- ✅ **상태 이상 아이콘**: 모든 상태 이상을 머리 위에 아이콘으로 표시
  - 석화: ◆, 기절: *, 수면: z/Z, 중독: P, 출혈: ~, 마나쉴드: ◇
  - 지속시간 카운트다운 표시
