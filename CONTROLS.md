# Game Controls

This document lists all the keyboard shortcuts and controls for Dungeon Crawler.

## General Controls
- **Movement**: `Arrow Keys` or `W`, `A`, `S`, `D`
- **Wait / Pass Turn**: `.`, `X`, `Z`, or `5`
- **Interact / Pick Up / Use Stairs**: `ENTER`
- **Attack Mode (Toggle)**: `SPACE` (Once active, press a direction key to attack)
- **Quick Slots**: `1` through `0` (Use bound skills or items)
- **Character Sheet**: `C` (View stats and allocate bonus points)
- **Inventory**: `I` (Open/Close inventory)
- **Quit Game (During Gameplay)**: `Q` or `ESC`

## Inventory & Menus
- **Navigate**: `Arrow Keys` or `W`, `S`
- **Switch Tabs**: `Left` / `Right` Arrow keys (Inside Inventory/Shop)
- **Use / Equip**: `E` or `ENTER`
- **Drop Item**: `X`
- **Close Menu**: `ESC`, `Q`, or the menu's toggle key (`I`, `C`)

## Character Sheet (Stat Point Allocation)
- **Select Stat**: `UP` / `DOWN` Arrow keys or `W`, `S`
- **Allocate Point**: `RIGHT` Arrow key, `D`, `+`, `=`, or `ENTER`
- **Close**: `C`, `ESC`, or `Q`

## Hall of Fame / Hall of Dishonor
- **Navigate**: `UP` / `DOWN` Arrow keys or `W`, `S`
- **View Details**: `ENTER`
- **Back to Menu**: `B`

# 조작법 (Korean)

이 문서는 던전 크롤러의 모든 조작법과 단축키를 설명합니다.

## 일반 조작
- **이동**: `방향키` 또는 `W`, `A`, `S`, `D`
- **대기 / 턴 넘기기**: `.`, `X`, `Z`, 또는 `5`
- **상호작용 / 아이템 줍기 / 계단 이용**: `ENTER`
- **공격 모드 (토글)**: `SPACE` (활성화 후 방향키를 눌러 공격)
- **퀵슬롯**: `1` ~ `0` (등록된 스킬이나 아이템 사용)
- **캐릭터 정보**: `C` (스탯 확인 및 포인트 배분)
- **인벤토리**: `I` (열기/닫기)
- **게임 종료 (플레이 중)**: `Q` 또는 `ESC`

## 인벤토리 및 메뉴
- **내비게이션**: `방향키` 또는 `W`, `S`
- **탭 전환**: `왼쪽` / `오른쪽` 방향키 (인벤토리/상점 내부)
- **사용 / 장착**: `E` 또는 `ENTER`
- **아이템 버리기**: `X`
- **메뉴 닫기**: `ESC`, `Q` 또는 해당 메뉴 단축키 (`I`, `C`)

## 캐릭터 정보창 (스탯 투자)
- **항목 선택**: `위` / `아래` 방향키 또는 `W`, `S`
- **스탯 투자**: `오른쪽` 방향키, `D`, `+`, `=`, 또는 `ENTER`
- **닫기**: `C`, `ESC` 또는 `Q`

## 명예의 전당 / 불명예의 전당
- **내비게이션**: `위` / `아래` 방향키 또는 `W`, `S`
- **상세 정보 보기**: `ENTER`
- **메뉴로 돌아가기**: `B`
