# 던전 크롤러 (Dungeon Crawler)

## 1. 게임 설명

이 프로젝트는 터미널 환경에서 실행되는 클래식한 로그라이크 던전 크롤러 게임입니다. 플레이어는 절차적으로 생성되는 던전을 탐험하며 강력한 몬스터들과 싸우고, 숨겨진 아이템과 장비를 찾아 캐릭터를 성장시켜야 합니다. 스킬북을 획득하여 새로운 능력을 배우거나 기존 스킬을 강화할 수 있으며, 매번 새로운 모험을 즐길 수 있습니다.

## 2. 주요 기능

*   **절차적으로 생성되는 던전**: 플레이할 때마다 던전의 구조, 몬스터, 아이템 배치가 달라집니다.
*   **턴 기반 시스템**: 플레이어와 몬스터가 한 턴씩 행동을 주고받는 전략적인 전투 방식입니다.
*   **아이템 및 장비**: 무기, 방어구, 소모품 등 다양한 아이템을 획득하여 캐릭터를 강화할 수 있습니다.
*   **스킬 시스템**: 던전에서 스킬북을 획득하면 즉시 스킬을 배우거나 기존 스킬의 레벨을 올릴 수 있습니다.
*   **영구 저장**: 게임 진행 상황이 자동으로 저장되어 언제든지 이어서 플레이할 수 있습니다.

## 3. 조작법 (Manual)

### 이동
-   **방향키, h, j, k, l**: 상하좌우 이동
-   **y, u, b, n**: 대각선 이동

### 게임 메뉴 및 상호작용
-   **i**: 인벤토리 열기/닫기
-   **m**: 전체 메시지 로그 보기
-   **r**: 몬스터 시체에서 아이템 루팅
-   **q**: 게임 저장 및 종료

### 인벤토리 내 조작
-   **방향키**: 아이템 선택
-   **a, b, c, d, z**: 아이템 탭 전환 (일반, 장비, 스크롤, 스킬북, 전체)
-   **e**: 장비 장착 / 스킬 퀵슬롯 등록
-   **u**: 소모품 아이템 사용
-   **R**: 선택한 아이템 버리기
-   **1~5**: 아이템 퀵슬롯에 등록
-   **6~0**: 스킬 퀵슬롯에 등록

## 4. 아이콘 범례 (Icon Legend)

| 아이콘 | 의미         | 설명                                       |
| :----: | :----------- | :----------------------------------------- |
|   `@`  | 플레이어     | 당신의 캐릭터입니다.                       |
|  `g, o, s, D`  | 몬스터       | 각각 고블린, 오크, 슬라임, 드래곤을 나타냅니다. |
|   `%`  | 몬스터 시체 | 아이템을 가지고 있을 수 있습니다. (`r` 키로 루팅) |
|   `#`  | 벽           | 지나갈 수 없는 지형입니다.                 |
|   `<`  | 시작 지점    | 현재 층을 시작한 위치입니다.               |
|   `>`  | 다음 층 입구 | 다음 층으로 내려가는 계단입니다.           |
|   `X`  | 잠긴 문      | 열쇠가 있어야 열 수 있는 다음 층 입구입니다. |
|   `+`  | 방 입구      | 다른 방으로 연결되는 문입니다.             |
|   `*`  | 아이템       | 바닥에 떨어져 있는 아이템입니다.           |
|   `T`  | 플레이어 무덤| 플레이어가 사망한 위치입니다.              |
