# 던전 크롤러 (Dungeon Crawler)

## 개요

이 프로젝트는 터미널 환경에서 실행되는 클래식한 로그라이크 던전 크롤러 게임입니다. 플레이어는 던전을 탐험하고, 몬스터와 싸우고, 아이템을 수집하며 캐릭터를 성장시켜야 합니다.

## 실행 방법

1.  필요한 라이브러리를 설치합니다.
    ```bash
    pip install readchar wcwidth
    ```

2.  게임을 시작합니다.
    ```bash
    python3 /home/dogsinatas/python_project/dungeon/Start.py
    ```

## 주요 기능

*   **절차적으로 생성되는 던전**: 매번 새로운 구조의 던전을 탐험할 수 있습니다.
*   **턴 기반 전투**: 플레이어와 몬스터가 차례대로 행동하는 전략적인 전투 시스템.
*   **아이템 및 장비**: 다양한 아이템과 장비를 획득하여 캐릭터를 강화할 수 있습니다.
*   **스킬 시스템**: 스킬북을 획득하여 새로운 스킬을 배우고 레벨업할 수 있습니다.
*   **저장 및 이어하기**: 게임 진행 상황을 저장하고 언제든지 이어서 플레이할 수 있습니다.
