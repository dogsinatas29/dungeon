이동 : 방향키 <br>
게임 실행 : python3 Start.py<br>

문자 바이트 수 때문에 문제가 발생하여 모두 영문 폰트를 기준으로 변경했습니다. <br>
r키는 아이템 루팅입니다. <br>
i키는 인벤토리 창입니다. <br>
  인벤토리에서 a,b,c,d,z로 인벤토리를 분할해 볼 수 있습니다. <br>
  장착은 e키입니다<br>
  이제 오른쪽 UI의 인벤토리 부분을 퀵 슬롯으로 바꾸고 물약과 스크롤등을 장착하게 할 예정입니다<br>
m키는 메시지 로그 창입니다 <br>
v키는 fog 토글입니다. <br>
이동은 방향키입니다. <br>

맵은 층마다 방이 딸려 있는 구조입니다<br>
'<'는 층의 Start point입니다. <br>
'>'는 다음층 게이트입니다.<br>
'+'은 방에 딸린 룸입니다.<br>
'*'는 아이템입니다. <br>

몬스터들은 이동형, 고정형, 회피형이 있습니다.<br>
인벤토리를 구성하고 나면 무기와 아이템 장착, 사용, 그리고 스킬의 구현에 들어갑니다. <br>
이게 끝나면 본격적으로 데이터 시트를 만들고 게임을 만들 예정입니다. <br>

게임을 시작하면 데이터 저장문제로 메인 메뉴의 저장 기능이 없습니다. <br>
추구 영구적 사망을 구현하는 과정에서 저장 기능과 캐릭터 이름 생성 기능을 다시 열어두겠습니다<br>
