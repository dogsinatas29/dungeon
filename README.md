# 던전 크롤러 (Dungeon Crawler)

## 1. 게임 설명

이 프로젝트는 터미널 환경에서 실행되는 클래식한 로그라이크 던전 크롤러 게임입니다. 플레이어는 절차적으로 생성되는 던전을 탐험하며 강력한 몬스터들과 싸우고, 숨겨진 아이템과 장비를 찾아 캐릭터를 성장시켜야 합니다. 스킬북을 획득하여 새로운 능력을 배우거나 기존 스킬을 강화할 수 있으며, 매번 새로운 모험을 즐길 수 있습니다.

## 2. 주요 기능

*   **절차적으로 생성되는 던전**: 플레이할 때마다 던전의 구조, 몬스터, 아이템 배치가 달라집니다.
*   **턴 기반 시스템**: 플레이어와 몬스터가 한 턴씩 행동을 주고받는 전략적인 전투 방식입니다.
*   **아이템 및 장비**: 무기, 방어구, 소모품 등 다양한 아이템을 획득하여 캐릭터를 강화할 수 있습니다.
*   **아이템 루팅**: 몬스터를 처치하고 남은 시체에서 아이템을 획득할 수 있습니다.
*   **다양한 스킬 시스템**:
    *   **스킬 습득**: 던전에서 스킬북을 획득하면 즉시 스킬을 배우거나 기존 스킬의 레벨을 올릴 수 있습니다.
    *   **발사체 스킬**: 방향을 지정하여 원거리의 적들을 타격하는 스킬입니다. (예: 파이어볼)
    *   **자가 시전 스킬**: 자기 자신에게 이로운 효과를 주는 스킬입니다. (예: 힐)
*   **영구 저장**: 게임 진행 상황이 자동으로 저장되어 언제든지 이어서 플레이할 수 있습니다.

## 3. 조작법 (Manual)

### 이동
-   **방향키, h, j, k, l**: 상하좌우 이동
-   **y, u, b, n**: 대각선 이동

### 게임 메뉴 및 상호작용
-   **i**: 인벤토리 열기/닫기
-   **m**: 전체 메시지 로그 보기
-   **r**: 몬스터 시체에서 아이템 루팅
-   **1~5**: 퀵슬롯의 아이템 사용
-   **6~0**: 퀵슬롯의 스킬 사용 (발사체 스킬의 경우 방향키로 조준)
-   **q**: 게임 저장 및 종료

### 인벤토리 내 조작
-   **방향키**: 아이템 선택
-   **a, b, c, d, z**: 아이템 탭 전환 (일반, 장비, 스크롤, 스킬북, 전체)
-   **e**: 선택한 장비 장착/해제, 소모품 사용, 스킬북 배우기
-   **R**: 선택한 아이템 버리기
-   **1~5**: 선택한 소모품/스크롤을 아이템 퀵슬롯에 등록
-   **6~0**: 선택한 스킬북에 해당하는 배운 스킬을 스킬 퀵슬롯에 등록

## 4. 아이콘 범례 (Icon Legend)

| 아이콘 | 의미         | 설명                                       |
| :----: | :----------- | :----------------------------------------- |
|   `@`  | 플레이어     | 당신의 캐릭터입니다.                       |
|  `g, o, s, D`  | 몬스터       | 각각 고블린, 오크, 슬라임, 드래곤을 나타냅니다. |
|   `%`  | 몬스터 시체 | 아이템을 가지고 있을 수 있습니다. (`r` 키로 루팅) |
|   `#`  | 벽           | 지나갈 수 없는 지형입니다.                 |
|   `<`  | 시작 지점    | 현재 층을 시작한 위치입니다.               |
|   `>`  | 다음 층 입구 | 다음 층으로 내려가는 계단입니다.           |
|   `X`  | 잠긴 문      | 열쇠가 있어야 열 수 있는 다음 층 입구입니다. |
|   `+`  | 방 입구      | 다른 방으로 연결되는 문입니다.             |
|   `*`  | 아이템       | 바닥에 떨어져 있는 아이템입니다.           |
|   `T`  | 플레이어 무덤| 플레이어가 사망한 위치입니다.              |

## 5. 실행 방법

1.  필요한 라이브러리를 설치합니다.
    ```bash
    pip install readchar wcwidth
    ```

2.  게임을 시작합니다.
    ```bash
    python3 /home/dogsinatas/python_project/dungeon/Start.py
    ```

## 6. 최근 변경 사항 (2025년 10월 9일)

*   **모듈 임포트 오류 해결**: `dungeon/player.py`, `dungeon/renderer.py`, `dungeon/trap.py`에서 `ModuleNotFoundError`를 유발하던 절대 경로 임포트를 상대 경로 임포트로 수정했습니다.
*   **데이터 파일 경로 수정**: `dungeon/data_manager.py`에서 아이템, 몬스터, 스킬, 함정 데이터 파일 경로가 `data/` 디렉토리를 올바르게 참조하도록 수정했습니다.
*   **몬스터 AI 이동 로직 구현**: 몬스터의 `move_type` (AGGRESSIVE, COWARDLY, STATIONARY)에 따라 플레이어를 추격하거나, 도망치거나, 움직이지 않도록 AI 이동 로직을 `dungeon/engine.py`에 구현했습니다. 몬스터가 함정을 밟았을 때의 처리 로직도 추가했습니다.
*   **아이템 루팅 로직 개선**: 플레이어가 'r' 키를 눌렀을 때 몬스터 시체에서 떨어뜨린 아이템과 맵에 직접 떨어진 아이템을 모두 루팅할 수 있도록 `dungeon/engine.py`의 루팅 로직을 개선했습니다.
*   **렌더링 우선순위 조정**: 몬스터 시체 위에 아이템이 있을 경우 아이템이 보이도록 렌더링 우선순위를 조정했습니다.
*   **인벤토리 기능 개선**: 
    *   `i` 또는 `I` 키로 인벤토리를 열고 닫을 수 있도록 토글 기능을 구현했습니다.
    *   인벤토리 내에서 `e` 키를 눌러 선택한 장비를 장착/해제하거나, 소모품을 사용하거나, 스킬북을 배울 수 있도록 했습니다.
    *   인벤토리 내에서 `1`~`0` 키를 눌러 선택한 아이템/스킬을 퀵슬롯에 등록할 수 있도록 했습니다.
*   **맵 상호작용 개선**: 
    *   플레이어가 `ITEM_TILE` 위로 이동하면 자동으로 아이템을 획득하도록 했습니다.
    *   플레이어가 `ROOM_ENTRANCE` 위로 이동하면 다음 방으로 이동하도록 로직을 추가했습니다.
*   **`Item` 객체 생성 오류 해결**: `Item.__init__()` `TypeError`를 해결하기 위해 `Item.from_definition()` 클래스 메서드를 사용하여 `Item` 객체를 생성하도록 수정했습니다.
