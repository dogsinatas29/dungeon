나의 역할 
게임 전체 개발 관리 및 디버그 결과를 시니어 프로그래머인 웹 제미나이에게 보고 

웹 제미나이
시니어 프로그래머로서 게임의 로직 설계 및 코드 제공, 결과 분석 및 개선 방안 제시

Gemini cli
주니어 프로그래머로서 시니어가 만든 코드를 바탕으로 실제  게임 코드를 작성

보고 방식
1. 구글 드라이브에 rclone을 사용하여 작업 내용을 sync : 홈디렉토리에서 아래 명령어 실행
rclone sync "/home/dogsinatas/python_project/dungeon" "gdrive:GOOGLE AI Studio/Roguelike/dungeon" --exclude '.git/**' --exclude '**/*.pyc' --exclude '__pycache__/**'
2. 해당 파일을 싱크 후 제미나이에게 구글 드라이브의 파일 업로드 
3. 실제 코드 변경후 실행하여 결과 및 로그 파일을 웹 제미나이에게 전달

웹 제미나이의 역할 
1. 구글 드라이브에서 싱크된 파일을 확인
2. 코드 분석 및 개선 코드 작성 
3. 코드를 나를 통해 Gemini cli에게 전달

Gemini cli의 역할
1. 시니어 프로그래머가 작성한 코드를 바탕으로 실제 게임 코드 작성
2. 작성된 코드를 실행하여 소스 코드를 작성 및 신규 소스 코드, 데이터.csv 파일 생성
